# codex

## 前回調査したバージョン

0.77.0 (tag: rust-v0.77.0)

## 今回調査したバージョン

0.79.0 (tag: rust-v0.79.0)

## 追加された機能・設定項目

- マルチ会話の「agent control」を追加し、セッションから他の会話を生成/メッセージできるようになった。
- app-server に `thread/rollback` を追加し、IDE クライアントが直近 N ターンを巻き戻せるようになった。
- キャッシュ済み Web Search のみを使う `web_search_cached` を追加。
- `codex exec resume` 後にモデル/JSON/サンドボックスなどのグローバル exec フラグを渡せるようになった。
- TUI に時刻/バージョン条件付きのアナウンス Tip 表示を追加（リポジトリ内 TOML で制御）。
- `[analytics] enabled=...` を追加し、分析送信の有効化/無効化を設定可能にした。

## 廃止された(または廃止予定の)機能・設定項目

- 特筆事項なし。

## その他

- apply_patch のパッチマーカーに前後の空白があっても許容するよう改善。
- TUI2 のストリーミング transcript がリサイズ時に再フローし、コピー時のソフトラップ保持が改善。

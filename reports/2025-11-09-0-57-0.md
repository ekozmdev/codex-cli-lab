# codex

## 前回調査したバージョン

0.56.0 (tag: rust-v0.56.0)

## 今回調査したバージョン

0.57.0 (tag: rust-v0.57.0)

## 追加された機能・設定項目

- TUI でスラッシュコマンドや履歴を Ctrl+N / Ctrl+P で巡回できるようになり、バックトラック時には `/status` などの冗長メッセージを自動的にスキップするようになった (#1994, #6335)。
- Unified Exec の出力を Freeform 形式に統合し、承認不要の安全なコマンドを自動実行できるようになった (#6233, #6380)。
- App Server の `account/read` と `Thread` エンドポイントが拡張され、追加メタデータが取得可能になったほか、ドキュメントにスレッド／ターンの初期化手順が追記された (#6336, #6338, #6377)。
- TypeScript SDK に `network_access` と `web_search` の各オプションが追加され、モデルの外部アクセス権限を指定できるようになった (#6367)。

## 廃止された(または廃止予定の)機能・設定項目

- Unified Exec を有効化すると旧来の shell ツールが自動的に無効化されるようになり、実行経路が一本化された (#6345)。

## その他

- 長時間実行コマンドのタイムアウト判定が改善され、不要なリトライを避けるようになった (#6340, #6349)。
- Cloudflare 403 エラー時の HTML 出力をユーザーフレンドリーなメッセージに差し替えた (#6252)。
- Windows Subsystem for Linux 環境でのアップデート処理が Windows パスを正規化するようになり、更新失敗が減少した (#6097, #6086)。

# Codex CLI 0.58.0 リリースサマリ

- リリース日: 2025-11-13
- 前バージョン: 0.57.0

## ハイライト

1. **GPT-5.1 ファミリーへの移行支援**
   - コアのプロンプト定義が `gpt-5.1` ベースの新しいガイドラインに刷新され、Codex CLI のデフォルト挙動が最新モデル向けに最適化された。【F:reports/2025-11-13-0-58-0/report.md†L8-L12】
   - TUI には GPT-5.1 系モデルへの移行を案内する専用プロンプトが追加され、ユーザーはインタラクティブに移行の可否を選択できる。【F:reports/2025-11-13-0-58-0/report.md†L13-L16】

2. **サンドボックス実行フローの拡張**
   - `codex sandbox` サブコマンドが再設計され、macOS Seatbelt / Linux Landlock / Windows 制限トークンの各サンドボックスを CLI から直接トリガーできるようになった。【F:reports/2025-11-13-0-58-0/report.md†L18-L22】
   - macOS 向けには拒否ログの収集、Windows 向けにはキャプチャ出力の整形など、環境別の運用改善が組み込まれた。【F:reports/2025-11-13-0-58-0/report.md†L22-L24】

3. **ユーザーシェル連携とツール統合の強化**
   - 新しい `shell_command` ツール呼び出しが追加され、ユーザーのログインシェルを経由したコマンド実行を記録付きで取り扱えるようになった。【F:reports/2025-11-13-0-58-0/report.md†L26-L28】
   - 実行結果は `<user_shell_command>` タグにまとめられ、会話履歴への埋め込みや後続フローでの扱いが容易になっている。【F:reports/2025-11-13-0-58-0/report.md†L28-L30】

4. **設定編集と UI 操作性の改善**
   - 設定モジュールに差分適用ベースの `ConfigEdit` API が入り、モデル選択・通知フラグ・MCP サーバー設定などを安全に更新可能になった。【F:reports/2025-11-13-0-58-0/report.md†L32-L35】
   - TUI のサスペンド／レジューム制御やジョブ管理が見直され、SIGTSTP 後の状態復元やオーバーレイの描画が安定化した。【F:reports/2025-11-13-0-58-0/report.md†L35-L37】

## その他の変更

- プロトコル TS ジェネレーターは削除され、Rust 実装にフォーカスする構成へ整理された。
- Windows サンドボックスの監査ロジックが拡張され、`audit.rs` によるイベント収集が強化された。
- CLI とドキュメントの依存バージョンが最新 Rust エコシステムに追従した。

# codex

## 前回調査したバージョン

0.55.0 (tag: rust-v0.55.0)

## 今回調査したバージョン

0.56.0 (tag: rust-v0.56.0)

## 追加された機能・設定項目

- より軽量な新モデル GPT-5-Codex-Mini が追加され、コスト効率を重視した推論が選択できるようになった。
- App Server のエクスポート機能とスレッド／ターン API が v2 名前空間に対応し、新しい通知やログインフロー API（`account/login/*` など）を利用可能になった (#6212, #6214, #6183, #6288)。
- TypeScript SDK に `modelReasoningEffort` オプションが追加され、推論レベルの制御ができるようになった (#6237)。

## 廃止された(または廃止予定の)機能・設定項目

- 特筆事項なし。

## その他

- CLI が PID 1 で実行される環境でも動作するように初期化処理が調整された (#4200)。
- トークン更新のリトライ挙動が改善され、「Re-connecting」が頻発する状況を抑制した (#6231)。
- gpt-5-codex モデルが明示的に要求されない限りコミットを改変しないようガイドラインを更新した (#6333)。

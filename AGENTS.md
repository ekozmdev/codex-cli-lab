このリポジトリはOpenAIのcodex-cliに関する情報を集めたり、私の今の設定を記録するためのリポジトリです。

このリポジトリをクラウドcodeßxで動かす場合は以下のような手順で行います

特定の言語のセットアップは必要ないです

https://github.com/openai/codex/releasesの情報を確認する

このリポジトリのREADMEに書かれた最新バージョンと比較して新しいバージョンが公開されている場合は以下の手順で
変更点をまとめたレポートを作成します

https://github.com/openai/codex.git
をチェックアウトします。

タグの一覧を確認します

git diffで1つ前のバージョンと現在の最新バージョンのdiffを調べます。

変更内容をまとめて

/reports/yyy-MM-dd-x-y-z.mdというフォルダ名を作成し、その下にレポートを作成してください。
x-y-zはバージョンです。

レポートには細かいバグの修正に関する内容はいらないです。
新しい機能や設定項目に注目してください、


最後にのリポジトリのREADMEのバージョン情報を更新します。

```
- 最新バージョンは`0.57.0`です(2025/11/15)
```

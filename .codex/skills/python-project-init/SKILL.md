# Pythonプロジェクト初期化

## 一通りの初期化フロー
- `references/initialize-project.md` を参照して一連の手順を進める。

## 関連スキル
- uv の初期化は `uv-init-project` を使う。
- Ruff の導入は `ruff-setup` を使う。
- Ruff の運用は `ruff-operations` を使う。
- 依存管理は `uv-manage-dependencies` を使う。
